div 是一个标准的块级元素。一个块级元素会新开始一行并且尽可能撑满容器。 其他常用的块级元素包括 p 、 form 和HTML5中的新元素： header 、 footer 、 section 等等。
这是static。static 是默认值。任意 position: static; 的元素不会被特殊的定位。 一个 static 元素表示它不会被“positioned”，一个 position 属性被设置为其他值的元素表示它会被“positioned”。
这是relative。这个元素是相对定位的。如果它是 position: static; ，那么它的绝对定位子元素会跳过它直接相对于body元素定位。
这是relative1. relative 表现的和 static 一样，除非你添加了一些额外的属性。
这是relative2. 在一个相对定位（position属性的值为relative）的元素上设置 top 、 right 、 bottom 和 left 属性会使其偏离其正常位置。 其他的元素的位置则不会受该元素的影响发生位置改变来弥补它偏离后剩下的空隙。

这是absolute。这个元素是绝对定位的。它相对于它的父元素定位。
